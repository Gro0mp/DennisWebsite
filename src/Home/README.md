# Dennis Wong - Portfolio

My personal portfolio website showcasing my projects, skills, and experience as a Full Stack Developer.

## About

This is my portfolio site featuring an interactive 3D background and smooth scroll animations. It displays my work experience, technical skills, and featured projects in a modern, engaging interface.

## Features

- **Interactive 3D Background** - Three.js animated room scene with floating spheres
- **Responsive Design** - Mobile-friendly with hamburger menu
- **Smooth Scroll Animations** - Elements reveal as you scroll
- **Project Showcase** - Highlights key projects with tech stacks
- **Contact Form** - Integrated with EmailJS for direct messaging
- **Work Experience Section** - Professional background and roles

## Tech Stack

**Frontend:**
- React + Vite
- TailwindCSS
- Three.js (React Three Fiber)
- React Three Drei

**Additional Libraries:**
- EmailJS (contact form)
- Leva (development controls)
- Maath (smooth animations)
